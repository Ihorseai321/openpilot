Fork of openpilot
=======================
本 fork 是基于官方 openpilot 代码修改而来，新增、修改过程中会遵循以下原则：

- 尽可能保持版本稳定
- 尽可能不使用选项开关
- 尽可能保留官方功能不被修改

Features
=======================
 * 简体中文界面
 * 新增部分已知车型指纹
 * 移除请连网更新的提示
 * 版本更新时可以在重启时跳过编译更新
 * 支持本田 Bosch 系列车型踩油门时不退出系统
 * 启用官方默认关闭的自动变道辅助功能（车速 > 70 km/h）
